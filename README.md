# Fighting game created with HTML CANVAS & JAVASCRIPT

### How to play: Player 1 use keys A, W, D and Space. Player 2 (ref: enemy in JS) use Arrow keys.

1. Player 1

- A = left,
- W = up/jump
- D = right
- space = attack

2. Player 2

- ArrowLeft = left
- ArrowRight = right
- ArrowUp = up/Jump
- ArrowDown = attack
